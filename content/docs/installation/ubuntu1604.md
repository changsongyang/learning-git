# Ubuntu 16.04

## 安装

为了拿到最新版本的 git，增加仓库然后再安装：

```bash
sudo add-apt-repository ppa:git-core/ppa
sudo apt-get update
sudo apt-get install git
```

安装完成之后，验证版本：

```bash
git --version
git version 2.15.0
```

此时 git 是在 /usr/bin/ 目录下：

```bash
root@sky2:~# which git
/usr/bin/git
root@sky2:~# ls -l /usr/bin/git
-rwxr-xr-x 1 root root 1920512 Oct  4 03:50 /usr/bin/git
```
