# tag命令资料

记录git中tag命令的用法。

## tag命令用法

### 列出已有tag

	git tag
	git tag -l 'v1.4.2.*'

### 添加tag

添加轻量级标签：

	git tag -a v1.4

添加含附注类型的标签：

	git tag -a v1.4 -m 'my version 1.4'

推送到远程：

	git push origin v1.4

或者--tags推送全部：

	git push --tags

### 删除tag

删除本地tag：

	git tag -d v1.0.0

删除远程tag：

	git push origin :refs/tags/v1.0.0
